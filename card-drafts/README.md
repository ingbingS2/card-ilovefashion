# 슬라임 카드뉴스 시안 — 두 버전 비교용

제목 **‘먹지 마세요! 만지세요!’** · 넥슨 루디브리엄(장난감 마을) 픽셀 감성.
브라우저에서 열어 보고 **어느 방향으로 진행할지** 골라주세요.

## 두 버전

| 파일 | 방향 | 장수 |
|------|------|------|
| `version-a-trend.html` | **A안 · 트렌드 소개형** — 음식모형 슬라임이 뭔지, 왜 유행인지 처음 보는 사람도 이해되게 | 10장 |
| `version-b-vendors.html` | **B안 · 구매처 가이드형** — 표지 → 사과·버터·쫀득쿠·크로와상·호빵 슬라임별 **판매 업체 3곳** → CTA | 7장 |

## 여는 법
파일을 더블클릭하거나 브라우저로 드래그해서 엽니다. (인터넷 연결 필요 — 픽셀 폰트/내보내기 라이브러리 로드)

## PNG 내보내기
- 카드 아래 **[이 장 PNG]** = 한 장만 저장
- 상단 **[전체 PNG 내보내기]** = 전 장 연속 저장
- 저장 크기: **1080 × 1350 (4:5, 인스타 세로 규격)**

## 문구/내용 수정
각 HTML 파일 안 `window.CARDS = [ ... ]` 배열의 텍스트만 고치면 됩니다.
공통 디자인은 `pixel.css`, 렌더링·내보내기는 `cards.js`.

## ⚠️ B안 업체 정보는 플레이스홀더
`업체명 ① / @instagram / 특징 한 줄`은 **채워 넣기용 예시**입니다.
실제 상호를 지어내면 잘못된 정보가 되므로 비워뒀어요.
실제 슬라임 샵으로 채우고 싶으면 말씀해 주세요 — 웹 검색으로 찾아 채워드립니다.

## 인스타 핸들
`cards.js` 상단 `HANDLE = "@your_trend"` 를 실제 계정명으로 바꾸세요.
